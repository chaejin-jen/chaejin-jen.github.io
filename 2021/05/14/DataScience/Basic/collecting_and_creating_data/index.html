    HTML 태그의 구성

    HTML 태그 구성 요소 (2)

    • 태그 이름
    • 태그 속성(attribute)

    1. 태그 이름

      p, li, img 같은 것들

      태그를 상징하는 <> 기호 안에 태그 이름을 가장 먼저 넣어주게 됨

      <p>, <li>, <img> 처럼 말이죠.

    2. 속성

      모든 HTML 태그는 속성이라는 추가 정보를 가질 수 있음 (태그 이름이 아닌 것은 모두 속성이라 생각하면 됨)

      속성은 일반적으로 속성 이름과 속성 값을 하나의 쌍으로 갖게 됨 (예시: name="value")

      만약 HTML 태그가 <p>, </p> 태그처럼 둘로 나누어져 있다면, 시작 태그에 속성을 적어줌

      • 예시 1

        <li id="favorite">우유</li>

        위 <li> 태그에는 id라는 속성이 있고, 속성 값(value)는 "favorite"

      • 예시 2 (한 태그가 여러 속성을 가지는 경우)

        <img alt="logo" class="logo-img" src="/images/music/logo.png"/>

        위 <img> 태그에는 총 세 개의 속성이 있음
        alt 속성은 "logo"라는 값을, class라는 속성은 "logo-img"라는 값을, src라는 속성은 "/images/music/logo.png"라는 값을 각각 가짐

        어떠 종류가 있는지, 더 공부하고 싶다면? 참고링크

    HTML 태그의 구조

    데이터를 모으는 데 있어서 중요한 것은 태그의 구조이다.
    한 태그가 아닌, 여러 태그 사이의 관계에 대해 알아보자.

    한 페이지의 HTML 태그는 서로 연결되어 있다. 이 구조가 마치 가계도나 나무(트리)와 유사해서, 부모 관계라고 부르거나 트리 구조라도고 한다.

    예시 1 (간단)

    <ul>
    <li>커피</li>
    <li>녹차</li>
    <li>우유</li>
</ul>

    위 예시에선 <ul> 태그가 <li> 태그를 포함하므로 <ul> 태그가 부모, <li> 태그가 자녀로 트리 구조를 가진다.

    <ul> 태그와 <li> 태그의 기능보다 더 중요한 것은, **”<ul> 태그 아래에 세 개의 <li> 태그가 있고, 그 안에 데이터가 있다는 사실”**이다.

    <ul> 태그 아래의 첫 번째 <li> 태그는 ‘커피’라는 텍스트를 담고 있고, 두 번째 <li> 태그는 ‘녹차’를, 세 번째 <li> 태그는 ‘우유’를 담고 있다.

    즉, <ul>태그와 <li> 태그의 기능을 몰라도 데이터의 구조를 파악할 수 있다.

    예시 2 (좀 더 복잡)

    <!DOCTYPE html>
<html>
<head>
    <title>Sample Website</title>
</head>
<body>
<h2>HTML 연습!</h2>

<p>이것은 첫 번째 문단입니다.</p>
<p>이것은 두 번째 문단입니다!</p>

<ul>
    <li>커피</li>
    <li>녹차</li>
    <li>우유</li>
</ul>

<img src="https://upload.wikimedia.org/wikipedia/commons/thumb/4/45/A_small_cup_of_coffee.JPG/550px-A_small_cup_of_coffee.JPG">
</body>
</html>

    위 예시 코드의 구조를 파악해보면 다음과 같다.

    <html> 태그에 <head> 태그와 <body>태그가 있고,
    <body> 태그 안에는 <h2> 태그, <p>태그, <ul> 태그, <img> 태그가 있고,
    <ul> 태그 안에는 세 개의 <li> 태그가 있다.

    CSS : 웹 페이지 스타일링

    HTML 코드에서 데이터를 추출하기 위해 웹 페이지 스타일링 원리를 알아보자

    HTML 코드에서 데이터를 골라내는 작업은 특정 태그를 선택하여 이뤄지므로,

    웹 페이지를 꾸미는 스타일링 원리와 비슷하기 때문이다.

    CSS : 웹 페이지를 꾸미는 언어

    • CSS를 사용하면 HTML 태그에 색, 폰트, 크기, 여백 등을 설정 가능

    [예시] CSS

        li {
    color: blue;
    font-size: 20px;
    }

    분류(2)

    • ‘HTML 태그를 선택하는 코드’
    • ‘스타일을 지정하는 코드’

    1. HTML 태그를 선택하는 코드

    위의 CSS 코드에서 ‘HTML 태그를 선택하는 코드’ : li 

        li {
    }

    이후에 자세히 살펴보겠지만, 이 코드는 **”모든 <li> 태그를 대상으로 스타일을 적용하라”**는 의미
    이렇게 스타일 적용의 범위를 정하는 코드를 **CSS 선택자(CSS Selector)**라 함

    여기에는 .favorite, #navbar 등 다양한 표현이 들어갈 수 있음
    CSS 선택자는 웹 스크레이핑에 매우 중요하기 때문에 나중에 다시 공부…

    2. 스타일을 지정하는 코드

        color: blue;
    font-size: 20px;
    • 첫 줄 color: blue;은 색상을 파란색(blue)으로 바꾸라는 의미
    • 두 번째 줄 font-size: 20px;은 글자 크기를 20px로 하라는 의미

    참고 링크 : css reference

    결국 두 코드를 합쳐보면

        li {
      color: blue;
      font-size: 20px;
    }

    “모든 li 태그의 색깔은 파란색으로, 크기는 20px로 하라” 라는 코드

    CSS 선택자

    첫 번째 CSS 선택자: 태그 이름

        li {
      color: blue;
      font-size: 20px;
    }

    위 코드의 의미 : “모든 <li> 태그의 색깔은 파란색으로, 크기는 20px로 하라”

    원리 1 : CSS 선택자로 ‘태그의 이름’을 쓰면, 그 태그 모두에 스타일이 입혀집니다.

    예를 들어 CSS 선택자에 해당하는 li를 p로 바꾸면,

        p {
      color: blue;
      font-size: 20px;
    }

    모든 <p> 태그에 스타일이 적용됨
    (“모든 <p> 태그의 색깔은 파란색으로, 크기는 20px로 하라”)

    CSS 코드가 없는 예시

    CSS 코드가 적용된 예시

    결론 : CSS 선택자로 태그의 이름을 쓰면, 그 태그 모두에 스타일이 입혀진다

    두 번째 CSS 선택자: 아이디 (id)

    모든 태그가 아닌, 단 하나의 태그에만 스타일을 입히고 싶을 경우, **’아이디(id)’**라는 속성을 사용

    아래와 같은 식으로, HTML 태그에 아이디(id)를 지정 가능

        <li id="coffee">커피</li>
    <li id="green-tea">녹차</li>
    <li id="milk">우유</li>

    아이디를 CSS 선택자로 활용하면, 그 아이디를 가진 태그에만 스타일이 입혀짐
    아이디를 상징하는 기호는 #인데, #coffee라고 하면 아이디가 coffee인 태그만 선택됨

        #green-tea {
      color: blue;
      font-size: 20px;
    }

    위의 코드 의미 : “아이디가 green-tea인 태그의 색깔을 파란색으로, 크기를 20px로 하라”

    주의!! 동일한 아이디를 여러 태그가 가질 수 없음! (하나의 아이디는 HTML 코드 전체에 단 한 번만 나와야 함)

    세 번째 CSS 선택자: 클래스(class)

    아이디는 딱 하나의 HTML 태그만 선택할 수 있음

    여러 태그에 동시에 스타일을 입히려면, **’클래스(class)’**라는 속성 사용

    클래스는 여러 HTML 태그에 공통으로 적용 가능

        <li class="favorite">커피</li>
    <li>녹차</li>
    <li class="favorite">우유</li>

    클래스를 활용해서 스타일링하면, 여러 태그 동시에 스타일을 입힐 수 있음

    클래스를 상징하는 기호는 .인데, .favorite라고 하면 클래스가 favorite인 모든 태그를 선택할 수 있습니다.

        .favorite {
      color: blue;
      font-size: 20px;
    }

    위의 코드 의미 : **”클래스가 favorite인 태그의 색깔을 파란색으로, 크기를 20px로 하라”**\

    CSS 코드가 없는 예시

    CSS 코드가 적용된 예시

    하나의 태그는 아이디를 한 개만 가질 수 있다
    클래스는 여러 개를 가질 수 있다 (클래스 이름 사이에 공백을 추가)

        <p class="favorite preview">이것은 첫 번째 문단입니다.</p>

    Beautiful Soup

    파싱할 HTML 코드를 변수 html_code에 넣어줌
    (파이썬에서 여러 줄의 텍스트를 """와 """사이에 두면, 하나의 문자열로 인식함)

    from bs4 import BeautifulSoup

    <title>Sample Website</title>
</head>

# BeautifulSoup 타입으로 변환
soup = BeautifulSoup(html_code, 'html.parser')

li_tags = soup.select('li')

beverage_names = []

    bs4 라이브러리의 BeautifulSoup을 불러옴

    from bs4 import BeautifulSoup

    불러온 BeautifulSoup에 HTML 코드를 넘겨주면, HTML 코드가 ‘BeautifulSoup 타입’으로 바뀜
    HTML을 파싱한다는 의미로 'html.parser'을 파라미터로 함께 넘겨주면 됨

    # BeautifulSoup 타입으로 변환

    ‘BeautifulSoup 타입’으로 바뀐 것 확인 완료

    CSS 선택자 더 알아보기

    실제 웹사이트의 복잡한 HTML 코드를 파싱하려면, CSS 선택자에 대해 좀 더 자세히 이해해야 함

    여러 CSS 선택자를 조합해서 사용 가능

    OR 연산

    쉼표 (,) 는 OR 연산을 의미하며 두 CSS 선택자 중 하나라도 해당되면 선택함

    예시 : 위 코드에서 class 이름이 "two"이거나, class 이름이 "four"인 태그를 선택 

    <p class="one">paragraph 1</p>
<p class="two">paragraph 2</p>
<p class="three">paragraph 3</p>
<p class="four">paragraph 4</p>
<p class="five">paragraph 5</p>

    .two, .four라고 쓰면 OR 연산 됨

    .two, .four {
    color: red;
}

    즉, 위 CSS 코드는 class 이름이 "two"이거나 "four"인 태그의 색상을 빨간색으로 지정하라는 의미

    CSS 코드가 없는 예시

    CSS 코드가 적용된 예시

    AND 연산

    두 CSS 선택자를 붙여쓰면, 두 조건에 모두 해당되는 요소만 선택

    예시 : "favorite" 클래스와 "private" 클래스를 모두 가진 태그를 선택

    <p class="favorite">paragraph 1</p>
<p class="favorite">paragraph 2</p>
<p class="favorite private">paragraph 3</p>
<p class="private">paragraph 4</p>
<p class="private">paragraph 5</p>

    .favorite.private라고 붙여서 작성하면 AND 연산 됨

    .favorite.private {
    color: red;
}

    즉, 위 CSS 코드는 "favorite" 클래스와 "private" 클래스를 모두 가진 태그의 색상을 빨간색으로 지정하라는 의미

    CSS 코드가 없는 예시

    [href="https://www.codeit.kr/"]href 속성의 값이 "https://www.codeit.kr/"인 태그

    1. OR 연산

    CSS 선택자의미
    #login, .left아이디가 login이거나 클래스가 left인 태그
    p, i모든 p 태그와 모든 i태그

    2. AND 연산

    두 CSS 선택자를 붙여쓰면 AND 연산이 됨 $\rightarrow$ 두 CSS 선택자에 모두 해당되는 요소만 선택

    CSS 선택자의미
    .favorite.privatefavorite 클래스와 private 클래스를 모두 가진 태그
    p.favoritefavorite 클래스를 가진 p 태그

    3. 중첩된 요소

    중첩된 태그를 선택하려면, 두 CSS 선택자를 띄어쓰면 됨
    $\rightarrow$ 첫 번째 CSS 선택자로 선택된 태그 내부에서, 두 번째 CSS 선택자를 찾음

    .favorite .privatefavorite 클래스를 가진 태그에 중첩된, private 클래스를 가진 태그

    import requests
from bs4 import BeautifulSoup
import pandas as pd

records = []
for year in range(2010, 2019):
    for month in range(1, 13):
        for week in range(0, 5):
            url = f"https://workey.codeit.kr/ratings/index?year={year}&month={month}&weekIndex={week}"

            soup = BeautifulSoup(response.text, 'html.parser')

            if len(soup.select('tr.row')) == 1:
                break
            else:
                program_tags = soup.select('tr.row')
                program_tags.pop(0)

                for tag in program_tags:
                    period = soup.select_one('#weekSelectBox [selected=\"selected\"]').get_text()
                    rank = tag.select_one('td.rank').get_text()
                    channel = tag.select_one('td.channel').get_text()
                    program = tag.select_one('td.program').get_text()
                    percent = tag.select_one('td.percent').get_text()
                    records.append([period, rank, channel, program, percent])


df = pd.DataFrame(data = records, columns=["period", "rank", "channel", "program", "rating"])

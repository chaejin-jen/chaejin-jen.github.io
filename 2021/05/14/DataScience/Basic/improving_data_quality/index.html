    통일성 (Conformity)

    데이터가 동일한 형식으로 저장되어 있어야 함

    형식 예시

    • 데이터 타입 : Int64, Float, String, Object, …
    • 단위 : $, kg, km, ml …
    • 포맷 : 서울 vs Seoul, 1,000 vs 1000, …


    정확성 (Accuracy)

    데이터가 정확해야 함

    데이터가 정확한지 확인하는 법 :

    이상점(Outlier)

    이상점 : 다른값과 심하게 동떨어진 값

    이상점 (Outlier)

    : 다른 값들과 너무 동떨어져 있는 데이터

    • 분석에 방해가 되면 제거
    • 의미 있는 정보라면 그냥 둠

    이상점 판단

    이상적을 판단하는 절대적인 기준은 없으며 다양한 방법이 있음 (데이터 사이언티스트의 역량임..)

    대표적 방법 : pandas 의 boxplot 에서의 기준

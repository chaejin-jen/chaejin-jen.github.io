    var로 선언

    동적언어 이므로 자료형을 선언할 필요 없음

    var a = 1;

    “기본 자료형”과 “객체(Object)” 두가지로 나뉨

    기본 자료형 설명
    Boolean 논리적 요소를 나타냄(True 와 False)
    Null 객체 값이 존재하지 않는다는 것을 의미
    Undefined 값을 할당하지 않음
    Number 숫자형
    String 문자형
    Symbol ECMAScript 6 에서 추가, 유일하고 변경 불가

    reduce

    배열.reduce((누적값, 현잿값, 인덱스, 요소) => { return 결과 }, 초깃값);

    • sort, every, some, find, findIndex, includes 모두 reduce로 구현 가능
    const numbers = [1,2,3,5,4];

// 누적덧셈
let acc = 0;
for(let i=0 ; i < numbers.length ; i++){
    acc = acc + numbers[i]
}
console.log(acc);

// reduce로 누적 덧셈 
const num = numbers.reduce(( acc, current)=>{
    return acc + current;
});

console.log(num)

    CSS를 사용하면 HTML 태그에 색, 폰트, 크기, 여백 등을 설정 가능

    <span>이 부분</span>을 두껍고 파랗게!

    <head>
  <!-- 중략 -->

  <style>
    span {
      font-weight: bold;
      color: blue;
    }
  </style>

</head>
<body>

  <span>이 부분</span>을 두껍고 파랗게!

</body>

    <head>

  <!-- 중략 -->
  <link rel="stylesheet" type="text/css" href="style.css"/>

  <span>이 부분</span>을 두껍고 파랗게!

</body>

    우선순위

    • 인라인 스타일 > id 선택자 > class 선택자 > 태그 선택자
    • 구체적일수록 높은 순위
    • 같은 우선순위라면 다음에(더 아랫줄) 오는 것이 덮어씀

    선택자

    • 기본적으로 CSS 선택자는 HTML 태그의 스타일을 지정하기 위해 사용

    • 하지만 추출할 데이터의 위치를 지정할 때도 활용 가능 (웹 페이지의 원리를 역이용)

    아이디와 클래스는 CSS 선택자로 자주 쓰이는 속성이기 때문에, #이나 . 처럼 쉽게 사용할 수 있는 문법 존재

    CSS 선택자를 더 많이 알고 싶다면 다음 링크 참고!
    W3Schools 링크

    CSS Selectors Cheatsheet

    종류

    tag 선택자

    문서 내 지명된 모든 태그에 적용

        li {
      color: blue;
      font-size: 20px;
    }

    위 코드의 의미 : “모든 <li> 태그의 색깔은 파란색으로, 크기는 20px로 하라”

    원리 1 : CSS 선택자로 ‘태그의 이름’을 쓰면, 그 태그 모두에 스타일이 입혀집니다.

    예를 들어 CSS 선택자에 해당하는 li를 p로 바꾸면,

        p {
      color: blue;
      font-size: 20px;
    }

    모든 <p> 태그에 스타일이 적용됨
    (“모든 <p> 태그의 색깔은 파란색으로, 크기는 20px로 하라”)

    결론 : CSS 선택자로 태그의 이름을 쓰면, 그 태그 모두에 스타일이 입혀진다

    class 선택자

    문서 내 다수 요소에 동일하게 적용될 수 있는 class 속성에 따라 적용

        <li class="favorite">커피</li>
    <li>녹차</li>
    <li class="favorite">우유</li>

    클래스를 활용해서 스타일링하면, 여러 태그 동시에 스타일을 입힐 수 있음

    클래스를 상징하는 기호는 .인데, .favorite라고 하면 클래스가 favorite인 모든 태그를 선택할 수 있습니다.

        .favorite {
      color: blue;
      font-size: 20px;
    }

    위의 코드 의미 : “클래스가 favorite인 태그의 색깔을 파란색으로, 크기를 20px로 하라”

    하나의 태그는 아이디를 한 개만 가질 수 있다
    클래스는 여러 개를 가질 수 있다 (클래스 이름 사이에 공백을 추가)

    예를 들어, <p> 태그에 "favorite", "preview" 두 개의 클래스를 모두 주고 싶다면 아래와 같이 작성 

        <p class="favorite preview">이것은 첫 번째 문단입니다.</p>

    id 선택자

    문서 내 유일한 요소에 적용될 수 있는 id 속성에 따라 적용

    • 하나 이상 사용 시 자바스크립트 등에서 오류 소지

    아래와 같은 식으로, HTML 태그에 아이디(id)를 지정 가능

        <li id="coffee">커피</li>
    <li id="green-tea">녹차</li>
    <li id="milk">우유</li>

    아이디를 CSS 선택자로 활용하면, 그 아이디를 가진 태그에만 스타일이 입혀짐
    아이디를 상징하는 기호는 #인데, #coffee라고 하면 아이디가 coffee인 태그만 선택됨

        #green-tea {
      color: blue;
      font-size: 20px;
    }

    위의 코드 의미 : “아이디가 green-tea인 태그의 색깔을 파란색으로, 크기를 20px로 하라”

    주의!! 동일한 아이디를 여러 태그가 가질 수 없음! (하나의 아이디는 HTML 코드 전체에 단 한 번만 나와야 함)

    attr 선택자

    태그의 속성과 그 값에 따라 적용

    • 더 자세히 보기 (MDN)
    • 예시
      : <img> 태그에서 흔히 쓰는 alt 속성 alt 속성은 이미지가 로딩되지 않을 경우를 대비해서, 이 이미지가 어떤 이미지인지 텍스트로 알려주는 역할을 함
      <img src='https://i.imgur.com/bY0l0PC.jpg' alt="coffee"/>
<img src='https://i.imgur.com/fvJLWdV.jpg' alt="green-tea"/>
<img src='https://i.imgur.com/rNOIbNt.jpg' alt="milk"/>

    속성 이름과 속성 값을 CSS 선택자로 활용할 가능. [name="value"]의 형태로 사용

        [alt="green-tea"] {
      width: 300px;
    }

    위의 코드 의미 : “alt 속성의 값이 "green-tea"인 태그의 가로 길이를 300px로 하라”

    CSS 적용 전 CSS 적용 후

    녹차에 대한 이미지만 가로 길이가 줄어들음

    조합

    실제 웹사이트의 복잡한 HTML 코드를 파싱하려면, CSS 선택자에 대해 좀 더 자세히 이해해야 함

    여러 CSS 선택자를 조합해서 사용 가능

    OR 연산

    쉼표 (,) 는 OR 연산을 의미하며 두 CSS 선택자 중 하나라도 해당되면 선택함

    예시 : 위 코드에서 class 이름이 "two"이거나, class 이름이 "four"인 태그를 선택 

    <p class="one">paragraph 1</p>
<p class="two">paragraph 2</p>
<p class="three">paragraph 3</p>
<p class="four">paragraph 4</p>
<p class="five">paragraph 5</p>

    .two, .four라고 쓰면 OR 연산 됨

    .two, .four {
    color: red;
}

    즉, 위 CSS 코드는 class 이름이 "two"이거나 "four"인 태그의 색상을 빨간색으로 지정하라는 의미

    AND 연산

    두 CSS 선택자를 붙여쓰면, 두 조건에 모두 해당되는 요소만 선택

    예시 : "favorite" 클래스와 "private" 클래스를 모두 가진 태그를 선택

    <p class="favorite">paragraph 1</p>
<p class="favorite">paragraph 2</p>
<p class="favorite private">paragraph 3</p>
<p class="private">paragraph 4</p>
<p class="private">paragraph 5</p>

    .favorite.private라고 붙여서 작성하면 AND 연산 됨

    .favorite.private {
    color: red;
}

    즉, 위 CSS 코드는 "favorite" 클래스와 "private" 클래스를 모두 가진 태그의 색상을 빨간색으로 지정하라는 의미

    중첩된 요소

    HTML 태그 안에 또 다른 HTML 태그가 있을 경우, 태그가 중첩되어 있다고 함

    일반적인 웹 페이지의 HTML 태그는 많이 중첩되어 있음
    때문에, 중첩에 대한 조건을 추가하는 경우가 많음

    예시 : 모든 <i> 태그가 아닌, <p> 태그 안에 중첩된 <i>태그만 가져오기

    (참고 : <i> 태그는 텍스트를 기울이는 이탤릭 효과를 의미)

    <i>디저트</i>
<p>
    <i>다쿠아즈</i>
    <i>마카롱</i>
    <i>케이크</i>
</p>

    "디저트"는 제외하고, "다쿠아즈", "마카롱", "케이크"만 가져오려하는 경우,

    띄어쓰기로 중첩을 표현 가능

    CSS 선택자를 p i라고 작성하면, <p>태그 안에 중첩된 <i>태그를 의미함

    p i {
    color: red;
}

    즉, 위 CSS 코드는 <p>태그 안에 있는 <i>태그만 빨간색으로 지정하라는 의미

    정리

    위에 CSS 선택자 복습 + 정리

    1. 태그 이름

      특정 태그 전체를 선택하려면 태그 이름을 사용

      CSS 선택자의미
      li모든 li 태그
      p모든 p 태그

    2. 아이디

      특정 아이디의 태그를 선택하려면, 아이디 앞에 #을 붙임

      CSS 선택자의미
      #coffee아이디가 coffee인 태그
      #tea-leaf아이디가 tea-leaf인 태그

    3. 클래스

      특정 클래스의 태그를 선택하려면, 클래스 앞에 .을 붙임

      CSS 선택자의미
      .favorite클래스가 favorite인 태그
      .reverse클래스가 reverse인 태그

    4. 속성

      속성의 이름과 값으로 태그를 선택하려면, [name="value"] 형식으로 적음

      CSS 선택자의미
      [alt="green-tea"]alt 속성의 값이 "green-tea"인 태그
      [href="https://www.codeit.kr/"]href 속성의 값이 "https://www.codeit.kr/"인 태그

    1. OR 연산

      쉼표 (,)를 활용하면 OR 연산이 됨 $\rightarrow$ 두 CSS 선택자 중 하나라도 해당되면 선택

      CSS 선택자의미
      #login, .left아이디가 login이거나 클래스가 left인 태그
      p, i모든 p 태그와 모든 i태그

    2. AND 연산

      두 CSS 선택자를 붙여쓰면 AND 연산이 됨 $\rightarrow$ 두 CSS 선택자에 모두 해당되는 요소만 선택

      CSS 선택자의미
      .favorite.privatefavorite 클래스와 private 클래스를 모두 가진 태그
      p.favoritefavorite 클래스를 가진 p 태그

    3. 중첩된 요소

      중첩된 태그를 선택하려면, 두 CSS 선택자를 띄어쓰면 됨
      $\rightarrow$ 첫 번째 CSS 선택자로 선택된 태그 내부에서, 두 번째 CSS 선택자를 찾음

      CSS 선택자의미
      .favorite .privatefavorite 클래스를 가진 태그에 중첩된, private 클래스를 가진 태그
      p .favoritep 태그 아래에 중첩된, favorite 클래스를 가진 태그

  background-color: yellowgreen;
  display: inline-block;
  padding: 24px;
}

    아래부터

    body > div > span { display: none; }

    body > div > span { visibility: hidden; }

    body > div > span { opacity: 0; }

    열의 갯수와 크기 지정

    grid-template-columns

    절대값으로

    .wrapper {
  grid-template-columns: 100px 200px 300px;
}

    비율 분할(fractions)

    .wrapper {
  grid-template-columns: 1fr 2fr 3fr 4fr;
}

    혼합 사용 

    .wrapper {
  grid-template-columns: 200px 1fr 3fr;
}

    반복 지정 

    .wrapper {
  grid-template-columns: repeat(3, 1fr);
}

    행의 갯수와 크기 지정

    grid-template-rows

    절대값으로

    .wrapper {
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows: 100px 200px 400px;
}

    상대값은 높이가 지정되어 있을 때 

    .wrapper {
  grid-template-columns: repeat(3, 1fr);
  height: 800px;
  grid-template-rows: 1fr 2fr 3fr;
}

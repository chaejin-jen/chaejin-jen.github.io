    인터랙티브 코드 편집기

    사용 목적 : Data analysis + Deep learning

    • Jupyter : 웹 기반 인터랙티브 코드 편집기의 표준
    • Colab : 구글에서 제공하는 클라우드 기반 인터랙티브 코드 편집기

    Jupyter

    • IPython(http://ipython.org) 커널을 기반으로 한 대화형 파이썬 셸
    • 일반적인 터미널 셸 + 웹 기반 데이터분석 Notebook 제공
    • 미디어, 텍스트, 코드, 수식등을 하나의 문서로 표현가능
    • 사실상의 데이터분석 Interactive Shell의 표준
    • Julia + Python + R

    cell 단위로실행 $\rightarrow$ 실행시점에 해당코드가 memory에 올라감

    jupyter 설치

    Anaconda 설치부터 되어 있어야 함

    conda install jupyter

    jupyter 실행

    jupyter notebook

    • http://localhost:8888/tree 주소로 jupyter가 실행됨
    • 터미널창의 token 정보를 사용

    주요 단축키

    • 실행 명령어 ctrl + enter , shift + enter, alt + enter

    edit mode – cell 안 편집 상태

    • 툴팁 표시하기 : Shift + Tab
    • 들여쓰기 사용하기 : ctrl + ] or ctrl + [
    • 셀나누기 : ctrl + shift + -

    command mode – cell 밖 상태

    • 아래셀이랑 합치기: shift + M
    • 셀 오려두기 : x
    • 셀 copy : c
    • 셀 붙여넣기: v or shift + v
    • 셀 지우기 : d, d
    • 셀 지우기 취소: z
    • Markdown 변환 : m,m
    • Code로 변환: y, y

    Colab

    • 구글이 개발 클라우드 기반의 jupyter notebook
    • 구글 드라이브 + GCP + Jupiter 등이 합쳐져서 사용자가 손 쉽게접근
    • 초반 여러 가지 모듈 설치의 장점을 가짐
    • 구글 드라이브의 파일을 업로드하여 사용 가능한 장점 가짐
    • VSCode 등과 연결해서 사용가능
    • V100 이상의 GPU를 무료로 쓸 수 있다는 장점을 가짐

    Colab pro을 사용할 경우 안정적인 colab의 활용이 가능해짐

    단축키

    jupyter 와 비슷한듯 다름

    Actions Colab Jupyter
    show keyboard shortcuts Ctrl/Cmd M H H
    Insert code cell above Ctrl/Cmd M A A
    Insert code cell below Ctrl/Cmd M B B
    Delete cell/selection Ctrl/Cmd M D DD
    Interrupt execution Ctrl/Cmd M I II
    Convert to code cell Ctrl/Cmd M Y Y
    Convert to text cell Ctrl/Cmd M M M
    Split at cursor Ctrl/Cmd M - Ctrl Shift -

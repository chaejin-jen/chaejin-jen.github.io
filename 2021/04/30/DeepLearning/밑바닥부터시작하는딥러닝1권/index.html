    퍼셉트론 (인공뉴런, 단순 퍼셉트론)

    퍼셉트론(perceptron) 알고리즘 : 프랑크 로젠블라트(1975) 고안, 신경망(딥러닝)의 기원

    퍼셉트론의 구조 = 신경망과 딥러닝의 중요한 아이디어

    퍼셉트론이란?

    일종의 함수이다

    • 입력 : 다수의 신호, 출력 : 하나의 신호

    • 여기서 신호란 flow가 있는 것으로, 흐름을 만들어 정보(0 또는 1)을 앞으로 전달

      동작원리 (예시를 통한)

    • 기호 의미

      • 입력 신호 : $x_1$, $x_2$
      • 출력 신호 : y
      • 가중치 : $w_1$, $w_2$

    • 그래프 해석

      • 원 : 뉴런 또는 노드
      • 입력 신호가 뉴런에 보내질 때, 각각 고유한 가중치가 곱해짐 ($w_1x_1$, $w_2x_2$)
      • 뉴런에서 보내온 신호이 총 합($w_1x_1 + w_2x_2$)이 정해진 한계(역치)를 넘어설 때만 1을 출력
        • 이를 뉴런을 활성화한다 라고 표현하기도 함 (활성화 함수 이름이 여기서 온 것)
        • 정해진 한계를 임계값 이라 하고 $\theta$ 로 표기

    • 수식 표현

      가중치의 의미

      퍼셉트론은 복수 입력 신호 각각에 고유한 가중치 부여함

    가중치는 각 신호가 결과에 주는 영향력을 조절하는 요소로 작용

    즉, 가중치가 클수록 해당 신호는 중요하다는 의미임

    알기 쉽게 비유하자면, 가중치는 전류에서 저항에 해당(작용은 반대로 함, 신호가 클수록 잘 흐르기 때문에)

    단순한 논리 회로

    참고 : 컴퓨터구조 - 05. 논리회로와 데이터 표현

    퍼셉트론에서 매개변수 값을 정하는 건 컴퓨터가아니라 사람
    즉, 진리표라는 학습 데이터를 직접 눈으로 보면서, 매개변수의 값을 도출함

    기계학습은 컴퓨터가 이 매개변수의 값을 정하는 작업을 자동으로 하도록 한 것
    따라서, 학습이란 적절한 매개변수 값을 정하는 작업이며, 사람은 퍼셉트론의 구조(모델)을 고민하고 컴퓨터에 학습할 데이터를 주는 일을 함

    AND 게이트

    입력 둘, 출력 하나

    두 입력이 모두 1일 때만 1 출력, 그 외에는 0을 출력

    NAND 게이트

    NAND = Not AND = ~AND

    AND 게이트를 구현하는 매개변수의 부호를 모두 반전하면 NAND 게이트가 됨

    OR 게이트

    퍼셉트론의 구조와 매개변수

    퍼셉트론은 AND, NAND, OR 논리 회로로 표현이 가능하며, 그 구조는 모든 게이트에서 똑같다.
    세 가지 게이트에서 다른 것은 매개변수(가중치와 임계값)의 값이다.

    퍼셉트론 구현하기

    간단한 구현 : AND 함수

    $x_1$과 $x_2$를 인수로 받는 AND 함수의 구현

    def AND(x1, x2):
    w1, w2, theta = 0.5, 0.5, 0.7 # 초기화
    tmp = x1*w1 + x2*w2           # 가중치를 곱한 입력의 총합
    if tmp <= theta:              # 임계값보다 큰지 작거나 같은지
        return 0
    else : # elif tmp > theta:
        return 1

    AND(0, 0) # 0을 출력
AND(1, 0) # 0을 출력
AND(0, 1) # 0을 출력
AND(1, 1) # 1을 출력

    가중치와 편향 도입 : $\theta$ $\rightarrow$ -b

    $\theta$ 를 -b 로 치환하면 퍼셉트론의 동작을 아래와 같이 표현 가능
    (b는 bias 편향 이라는 의미)

    import numpy as np
x = np.array([0, 1])      # 입력
w = np.array([0.5, 0.5])  # 가중치
b = -0.7                  # 편향
w*x # array([0. , 0.5])
np.sum(w*x) # 0.5
np.sum(w*x) + b # -0.19999999999999996
                # (대략 -0.2 로 부동소수점 수에 의한 연산 오차이다)

    가중치와 편향 구현하기

    퍼셉트론의 구조는 AND, NAND, OR 게이트에서 모두 같고,
    다른 것은 각 게이트에서의 매개변수(가중치와 임계값)의 값임을 확인해보자!

    AND 게이트

    def AND(x1, x2):
    x = np.array([x1, x2])      # 입력
    w = np.array([0.5, 0.5])  # 가중치
    b = -0.7                  # 편향 $\theta$를 -b로 치환
    tmp = np.sum(w*x) + b     # 가중치를 곱한 입력의 총합
    if tmp <= 0:
        return 0
    else :
        return 1

    가중치($w_1$, $w_2$) : 각 입력 신호가 결과에 주는 영향력(중요도)를 조절하는 매개변수

    편향(b) : 뉴런이 얼마나 쉽게 활성화 (결과로 1을 출력)하느냐를 조정하는 매개변수

    예를 들어, b=-0.1 일때와 b=-20 일 때를 비교해보면, 가중치를 곱한 입력의 총합이 각각 0.1과 20을 초과할 때만 활성화 됨

    (문맥에 따라 가중치와 편향을 한번에 가중치라고 부르기도 함)

    내 해석 : 가중치($w_1$, $w_2$)는 $x_1$ 과 $x_2$ 의 증감률, 편향(b)는 1에 근접한 정도

    NAND, OR 게이트

    def NAND(x1, x2):
    x = np.array([x1, x2])      # 입력
    w = np.array([-0.5, -0.5])  # 가중치 AND 와 부호 반대
    b = 0.7                     # 편향 AND와 부호 반대
    tmp = np.sum(w*x) + b       # 가중치를 곱한 입력의 총합
    if tmp <= 0:
        return 0
    else :
        return 1

    def OR(x1, x2):
    x = np.array([x1, x2])      # 입력
    w = np.array([0.5, 0.5])    # 가중치
    b = -0.2                    # 편향
    tmp = np.sum(w*x) + b       # 가중치를 곱한 입력의 총합
    if tmp <= 0:
        return 0
    else :
        return 1

    퍼셉트론의 한계

    XOR 게이트

    XOR 게이트 : 베타적 논리합이라는 회로
    $x_1$과 $x_2$ 중 한쪽이 1일 때만 1 출력 (‘배타적’ : 자기 외에는 거부)

    지금까지의 퍼셉트론으론 XOR 게이트를 구현 불가!

    구현불가 증명

    구현이 불가함을 증명하기위해, 퍼셉트론을 시각화해보자.

    • OR 게이트 시각화
      (b, $w_1$, $w_2$) = (-0.5, 1.0, 1.0) 일 때, OR 게이트의 진리표를 만족함

      이때의 퍼셉트론은 아래 식으로 표현 가능

      위 식은 직선의 방정식으로 아래 그림 같이 표현이 가능

      OR 게이트는, ($x_1$, $x_2$) = (0,0) 일 때 0을 출력하고 (0, 1), (1, 0), (1, 1)일 때는 1을 출력함
      그림에서는 0을 원($\bigcirc$), 1을 삼각형($\triangle$)으로 표시함

      OR 게이트를 만들려면 위 그림에서 $\bigcirc$ 와 $\triangle$ 을 직선으로 나눠야 함

    • XOR 게이터 시각화

      $\bigcirc$과 $\triangle$는 XOR 게이트의 출력을 나타냄
      $\bigcirc$과 $\triangle$는 직선 하나로 나눌 수 없음

      XOR 게이트 구현 불가함

    선형과 비선형

    직선 이라는 제약조건을 없애면, 즉 “곡선” 이라면 $\bigcirc$과 $\triangle$ 를 나눌 수 있음

        2. 산점도(Scatter Plot)

        3. 히스토그램(Histogram)

        정답 : 히스토그램 (히스토그램을 그리면 고양이들의 몸무게 분포를 범위별로 파악 가능)

      1. 막대 그래프(Bar Graph)

      2. 산점도(Scatter Plot)

      3. 파이 그래프(Pie Graph)

      KDE Plot

      대부분의 확률은 정규 분포를 따른다. 그러나, 실제 가진 데이터를 확인해보면 그렇지 않다.

      • 실험이나 관찰을 통하여 수집된 자료집단의 확률분포는 대부분 정규분포를 따름 (참고:정규분포)

        • 정규분포는 자연과학 현상은 물론 사회과학 현상을 분석할 때 가장 빈번하게 활용되는 확률분포

      • 실제로 직접 모은 데이터의 분포를 확인해보면 울룩불룩 굴곡이 있음

        • 키, 강수량, 속도 등은 딱딱 나눠 떨어지는 수가 아닌, 무수한 수로 이루어졌기 때문 (무수한 데이터를 모으면 정규분포을 띄게 됨)

        • df['Height'].value_counts().sort_index().plot()

      • KDE(Kernel Density Estimation)

        가지고 있는 데이터를 가지고, 추측하여 정규분포 형태를 띄게 함

      중간값

      어떤 경우 중간값이 나을까?

      잘못 되거나 큰 데이터가 있을 경우 (평균값은 그런 데이터에 대해서 영향을 많이 받음)

      어떨 경우 평균값이 나을까?

      상관 계수(Correlation Coefficient)

      [참고] pandas merge 함수 설정값

      pd.merge(left, right, # merge할 DataFrame 객체 이름
      how=’inner’, # left, rigth, inner (default), outer
      on=None, # merge의 기준이 되는 Key 변수
      left_on=None, # 왼쪽 DataFrame의 변수를 Key로 사용
      right_on=None, # 오른쪽 DataFrame의 변수를 Key로 사용
      left_index=False, # 만약 True 라면, 왼쪽 DataFrame의 index를 merge Key로 사용
      right_index=False, # 만약 True 라면, 오른쪽 DataFrame의 index를 merge Key로 사용
      sort=True, # merge 된 후의 DataFrame을 join Key 기준으로 정렬
      suffixes=(‘_x’, ‘_y’), # 중복되는 변수 이름에 대해 접두사 부여 (defaults to ‘_x’, ‘_y’
      copy=True, # merge할 DataFrame을 복사
      indicator=False) # 병합된 이후의 DataFrame에 left_only, right_only, both 등의 출처를 알 수 있는 부가 정보 변수 추가

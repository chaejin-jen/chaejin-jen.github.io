    Boolean bool

    Complex complex64, complex128, complex

    Object object

    • Float
      • Float is always a double precision floating point value (64 bits).
      • longfloat represents large precision floats. Its size is platform dependent.
    • Complex
      • The real and complex elements of a complex64 are each represented by a single precision (32 bit) value for a total size of 64 bits.
    • Strings
      • Unicode is always UTF32 (UCS4)
    • Object
      • Represent items in array as Python objects.

    Hierarchy of types used in NumPy

    There are 21 “built-in” (static) data-type objects

    New (dynamic) data-type objects are created to handle

    1. Alteration of the byteorder
    2. Change in the element size (for string, unicode, and void built-ins)‏
    3. Addition of fields
    4. Change of the type object (C-structure arrays)

    인덱싱

    DataFrame 인덱싱을 하는 방법

    iloc 와 loc

    iloc : 위치로 인덱싱

    파이썬 문자열, 리스트 인덱스에서 익숙한 표현

    위치로 인덱싱하기 기본 형태 단축 형태
    하나의 row 위치 df.iloc[8]
    row 위치의 리스트 df.iloc[[4, 5, 3]]
    row 위치의 리스트 슬라이싱 df.iloc[2:5] df[2:5]
    하나의 column 위치 df.iloc[:, 3]
    column 위치의 리스트 df.iloc[:, [3, 5, 6]]
    column 위치의 리스트 슬라이싱 df.iloc[:, 3:7]

    loc : 이름으로 인덱싱

    Dataframe 에서 각 인덱스엔 문자형태도 올 수 있기 때문에, 인덱스 이름을 기준으로 인덱스 가능

    이름으로 인덱싱하기 기본 형태 단축 형태
    하나의 row 이름 df.loc["row4"]
    row 이름의 리스트 df.loc[["row4", "row5", "row3"]]
    row 이름의 리스트 슬라이싱 df.loc["row2":"row5"] df["row2":"row5"]
    하나의 column 이름 df.loc[:, "col1"] df["col1"]
    column 이름의 리스트 df.loc[:, ["col4", "col6", "col3"]] df[["col4", "col6", "col3"]]
    column 이름의 리스트 슬라이싱 df.loc[:, "col2":"col5"]

    데이터 사이언스란?

    데이터를 다루는 모든 일! 데이터속에서 유의미한 정보, 인사이트 발견!

    데이터 사이언스 단계

    일반적으로 다음과 같음

    출처 : 코드잇

    1. 문제 정의하기

    : 해결하고자 하는 문제를 정의 (What, When, How)

    • 목표 설정
    • 기간 설정
    • 평가 방법 설정
    • 필요한 데이터 설정

    뭐가 문제인지, 언제까지 어떤 결과물을 얻을 것인지, 어떤 방식으로 데이터를 활용할 것인지 등을 설정

    즉, 목적을 갖고 데이터를 살펴서, 의미 있는 발견을 하자! = 인사이트 발견의 기본 자세

    • 필요 역량 : 산업 트렌드 분석, 커뮤니케이션 능력

    2. 데이터 모으기

    : 필요한 데이터를 모을 수 있는 방법을 찾기

    • 파일 읽고 쓰기
    • 자료 모으기
    • 웹 크롤링

    누군가 이미 데이터를 모았다면 제일 좋고 (그대로 사용), 공공 기관 등에서 배포한 자료 활용, 웹사이트에서 직접 데이터를 수집

    • 필요 역량 : 빡쎈 검색과… 간단한 코딩..

    3. 데이터 다듬기

    : 데이터의 퀄리티 $\uparrow$ $\rightarrow$ 의미 있는 분석

    • 데이터 관찰하기
    • 데이터 오류 제거
    • 데이터 정리하기

    수집한 데이터 자체의 문제점으로 인해 분석 자체가 불가능할 수도 있고, 혹은 분석을 하더라도 잘못된 결론으로 이어질 수도 있음

    • 필요 역량 : 인내심(노..가..다..), 수학/통계적 지식, 도메인 지식(=전문지식), 코딩(예시 : Python 에서 Pandas 사용)

    4. 데이터 분석하기

    : 준비된 데이터로부터 의미를 찾기

    • 데이터 파악하기
    • 데이터 변형하기
    • 통계 분석
    • 인사이트 발견
    • 의미 도출

    통계를 이용한 수치적 분석, 여러 그래프를 그리는 탐색($^{***}$EDA)

    처음 설계했던 방식대로 데이터를 활용해서 원하는 결과를 도출해야함!

    • 필요 역량 : 수학/통계적 지식, 도메인 지식(=전문지식), 코딩(시각화)

    $^{***}$ 참고 : EDA (Exploratory Data Analysis) 탐색적 데이터 분석

    5. 데이터 시각화 및 커뮤니케이션

    : 분석 결과를 다른 사람들에게 전달

    • 다양한 시각화
    • 커뮤니케이션
    • 리포트

    어떤 문제를 해결하려 했는지, 어떻게 데이터를 모았는지, 어떤 방식으로 어떤 인사이트를 얻었는지 등을 다른 사람들에게 전달하는 과정

    적절한 시각화를 통해 소통을 원활히 하자!

    • 필요 역량 : 커뮤니케이션, 그를 위한 시각화 툴 사용

    데이터 사이언스 도구

    R 과 파이썬

    • R : 통계와 시각화만을 위한 툴

      • 주로 연구쪽에 많이 쓰임

    • Python : 다양한 분야 사용

      • 여러곳에 접목하기 좋음
      • 원래 개발자들이 사용도 함
      • Numpy, Pandas, Tensorflow 같은 라이브러리 또는 프레임워크가 파이썬에서 사용가능해지면서 더 인기

    • 2019년 기준 Analystics, Data Science, Machine Learning Software로 Python을 65% 이상 사용

    출처 : https://www.kdnuggets.com/2019/05/poll-top-data-science-machine-learning-platforms.html

    모델간의 관계도

    • OneToOne

    • OneToMany

      • 예시 : 하나의 상점에 여러개의 메뉴가 있다

    • ManyToOne

    • ManyToMany

      OneToOne

    • 1:1로 맵핑

    • Ex) 사용자 모델, 회사모델

      • 사용자가 속한 회사 (오직 하나만 있을 때)

    OneToMany

    • 1:N

    • Ex) 제품에 달린 댓글들

      • 제품 하나에 대해 여러개의 댓글이 달림
      • 외부키 제품ID

    ManyToOne

    • OneToMany와는 반대의 경우
    • 좌측은 1번만 가능
    • Ex) a는 1번 자동차, b는 1,2번 자동차

    ManyToMany

    • N:M 의 관계

    • 좌측 우측 여러번 선택 가능

    • 태그를 구현하는 경우 많이 쓰임

    • Ex) 인스타그램 태그, 좋아요 등

      • A 게시물에 걸린 태그확인
      • #서울 을 태그한 게시물을 확인

    2. associate 추가 : ./models/Shops.js 수정

    • foreignKey : 'shop_id' : 여러 메뉴가 있을텐데, 그 구분자를 shop_id 로 하겠다는 것 (즉, “가게 id” 별 메뉴)
    • sourceKey : 'id' : shop_id 로 Shops 테이블의 id를 받음
    • onDelete : 'CASCADE' : Shops 테이블에서 row 가 삭제되면, ShopsMenu 에서도 삭제됨
    const moment = require('moment');

module.exports = (sequelize, DataTypes) => {

        Shops.hasMany( models.ShopsMenu , {
            as : 'Menu',
            foreignKey : 'shop_id',
            sourceKey : 'id',
            onDelete : 'CASCADE'
        });
    };

    // ..생략 ..
}
    ./models/Shops.js 
    const moment = require('moment');

module.exports = (sequelize, DataTypes) => {
    const Shops = sequelize.define('Shops',
        {
            id: { type: DataTypes.INTEGER, primaryKey: true, autoIncrement: true },
            name : { type: DataTypes.STRING , comment: '상점명' },
            address : { type: DataTypes.STRING , comment: '주소' },
            location : { type: DataTypes.STRING , comment: '상세주소' },
            phone : { type: DataTypes.STRING , comment: '전화번호' },
            open_time : { type: DataTypes.STRING , comment: '운영시간' },
            cell_phone : { type: DataTypes.STRING , comment: '핸드폰번호' }
        }
    );

    Shops.associate = ( models ) =>  {
        
        Shops.hasMany( models.ShopsMenu , {
            as : 'Menu',
            foreignKey : 'shop_id',
            sourceKey : 'id',
            onDelete : 'CASCADE'
        });
    };

    Shops.prototype.dateFormat = (date) => (
        moment(date).format('YYYY-MM-DD')
    );

    return Shops;
};

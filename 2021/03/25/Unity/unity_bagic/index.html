    [여기서 잠깐!] new 란?

    단순변수가 아니라 메모리를 할당할 때 new 사용

    c에서는 관리자가 없어서 new 해주고 delete 해줘야함

    c#에서도 객체를 생성할 때 new를 써줘야 함. (delete 해줄 필요는 없음)

    Student1

    using System.Collections;
using System.Collections.Generic;
using UnityEngine;

public class scrip1 : MonoBehaviour
{
    // Start is called before the first frame update
    void Start()
    {
        transform.Translate(5, 0, -8);
        // transform.Translate(-1, 0, 1);
        Debug.Log("Start x1");
    }

    public float mX = 5; // 강제로 0이 들어감
    public float mZ = -8; // 강제로 0이 들어감

    // Update is called once per frame
    void Update()
    {   
        if (mX == 0 && mZ == 0)
        {
            mX = 5;
            mZ = -8;
        }

        if (mX > -3.0f && mZ < 1.0f)
        {
            mX -= Time.deltaTime + 0.05f;
            mZ += Time.deltaTime + 0.01f;
            transform.position = new Vector3(mX, 0, mZ);
            Debug.Log("Update:" + Time.deltaTime);
        }

        else
        {
            if (mZ < 1.0f)
            {
                mZ += Time.deltaTime + 0.05f;
                transform.position = new Vector3(mX, 0, mZ);
                Debug.Log("Update:" + Time.deltaTime);
            }

            else
            {
                if (mX < -1.2f)
                {
                    mX += Time.deltaTime + 0.05f;
                    transform.position = new Vector3(mX, 0, mZ);
                    Debug.Log("Update:" + Time.deltaTime);
                }
            }
        }

    }
}

    Prefab

    • Prefab : 여러 컴포넌트로 이미 구성이 완성된, 재사용 가능한 Game Object

    • 사용 :

      • 하나의 Game Object를 여러 개 사용해야 할 경우 사용
      • 장점 : 나중에 그 속성이 일부 변경될 때, 프리팹의 속성만 바꿔주면 프리팹으로 생성된 게임오브젝트에는 모두 반영

    What is the difference between a prefab and a gameobject?

    Prefab 만들기

    방법 1

    1. 일단 오프젝트를 하나 완성

    2. Assets 폴더의 Prefabs 폴더에 드래그

      Prefab으로 생성된다.

    주의 : 꼭 Prefabs 폴더여야 함 (Assets 폴더 아래에 Create > Folder 로 만들면 됨)

    방법 2

    1. 빈 프리팹을 먼저 생성 : Prefabs 폴더에서 마우스 우클릭 > Create > Prefab
    2. 오브젝트를 할당 : 게임오브젝트를 해당 프리팹으로 드래그

    특징

    • 한번 프리팹으로 생성되면 Scene에서 삭제해도 Asset에 남아있음
    • Prefab을 선택하고 컴포넌트, 속성을 변경하면 Sence에 사용된 Prefab에 모두 적용됨
      • Sence에 적용된 오브젝트를 선택하여 속성을 변경할 경우,
        • Apply 버튼으로 전체 적용 가능
      • Sence에 적용된 오브젝트를 선택하여 속성을 변경하고 Apply를 누르지 않으면,
        • 수정된 개별 오브젝트에만 수정사항이 반영되며,
        • 프리팹이 재 변경되더라도 한번 개별변경된 속성은 프리팹의 속성을 따라가지 않음
    • Revert 버튼 : 수정된 개별 오브젝트를 다시 프리팹의 속성대로 복원

    Select버튼을 누르면 게임오브젝트가 아닌 프리팹을 선택할 것으로 변경되어,
    프리팹을 편집할 수 있게 된다.

    1. 유니티 에셋 스토어에서 다운로드 후 임포트 하기.2. 외부 사이트에서 다운로드 후 임포트 하기.
    1. 유니티 에셋 스토어에서 다운로드 후 임포트 하기.
    1. 에셋 스토어 사이트(https://assetstore.unity.com/)에 로그인.2. 에셋을 둘러본 후 [Add to My Assets] 버튼 클릭.
    1. 유니티 에디터로 돌아온 후, 상단 메뉴 Window → Asset Store 클릭해서 Asset Store 창 띄우기(혹은 Ctrl 9). 그 후 아래 형광표시된 버튼 클릭(Toggle Download Manager).
    1. 클릭 후 2에서 My Assets에 추가했던 에셋 다운로드하기.
      1. 다운로드가 완료되면, Download 버튼 왼쪽 Import 버튼이 활성화 됨.
        Import 버튼 클릭하면 다음과 같은 창이 나옴. 제외할 항목 확인 후 이상 없다면 Import 버튼 클릭.

    2. 프로젝트 뷰에 Import 된 에셋 확인.

    1. 외부 사이트에서 다운로드 후 임포트 하기.
    1. 사용할 에셋 다운로드 후 유니티 엔진의 메뉴바에서 Assets → Import Package → Custom Package 클릭.2. Import 할 파일 선택 후 열기 클릭.
    1. 제외할 항목 확인 후 이상 없다면 Import 버튼 클릭.

    실습 : 학생들 자리 배정

    1. 대기열에 학생들 중에서 한명을 선택할 수 있다 (색상 또는 높이 차이로 표시)
    2. 만약 학생을 중복 선택할 경우 마지막 1명만 선택
    3. 책상을 선택하면 학생을 그 곳으로 이동
    4. 학생은 이동 또는 대기 등의 여러 상황에서 항상 앞을 바라봐야 함(학생 앞뒤 방향 구분 가능하게 만들어서, 앞부분으로만 전진)
    5. 책상은 2인용으로, 책상 옆에 먼저 자리 잡은 학생이 없는 방향으로 이동해야 함.
    6. 책상에 자리 잡은 학생을 누르면 대기열로 돌아감

    public List<Vector3> mPs = new List<Vector3>();

void Update() {
    float tr = Time.deltaTime * 2.0f;

    if (mPs.Count > 0){
        Vector3 pg = mPs[0],
            pc = transform.position;

        if(pc.x > pg.x){pc.x = Mathf.Max(pg.x, pc.x -tr);}
        else { pc.x = Mathf.Min(pg.x, pc.x +tr);}
        if (pc.z > pg.z) {pc.z = Mathf.Max(pg.z, pc.z -tr);}
        else { pc.x = Mathf.Min(pg.z, pc.z +tr);}
        transform.position = pc;
        if(pc.x == pg.x && pc.z == pg.z){mPs.RemoveAt(0);}
    }
}

  //Instantiate Prefab

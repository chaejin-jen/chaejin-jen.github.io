    • 벡터를 투영분해
    • 투영분해 결과, 직선의 방정식을 벡터 연산으로 나타내기
    • 벡터의 선형독립과 벡터공간의 의미
    • 벡터를 벡터공간에 투영
    • 기저벡터가 바뀌었을 때 이에 해당하게 좌표 변환
    • 고윳값 분해의 정의
    • 행렬의 모양과 고윳값의 관계에 대한 성질 암기
    • 특잇값 분해의 정의
    • 특잇값 분해를 차원축소문제에 어떻게 응용할 수 있을까

    미적분

    • 함수, 역함수의 의미
      * 함수를 파이썬으로 구현하는 법
      * 함수의 그래프를 그리는 방법과 함수의 기울기를 구하는 방법
    • 데이터 분석에 많이 쓰이는 다양한 함수의 종류와 특징

    • 심볼릭 연산의 의미
    • SymPy로 미분과 적분을 하는 방법
    • 부정적분과 정적분을 구별
    • 벡터와 행렬로 이루어진 함수를 미분
    • 범함수의 의미와 범함수의 도함수를 구하는 공식

    사이파이(SciPy)로 최적화

    • 최적화 문제의 의미와 풀이법

      • 최적화 문제 : 함수의 값을 가장 크게 혹은 작게 만드는 입력변수의 값 찾기
      • 모든 데이터 분석은 주어진 기준에 가장 적합한 수식을 찾는다 (= 일종의 최적화 문제를 푸는 과정)

    • 그리드 서치 방법과 단점

      • 그리드 서치 (grid search) : 가장 기본적인 최적화 방법으로, 일정 범위의 입력변숫값들을 모두 계산
      • 단점 : 입력변수의 수가 많거나 함수 계산에 시간이 오래 걸리는 경우 불가능한 방법임

        $\rightarrow$ 가능한 한 적은 횟수로 함수를 계산하는 수치적 최적화 방법을 사용

    • 수치적 최적화 방법 중 최급강하법과 뉴튼 방법

      • 최대경사법(steepest gradient method) : 수치적 최적화 방법 중 가장 단순한 방법
      • 뉴튼 방법(Newton method) : 개선된 최대경사법

    • 라그랑주 승수법과 라그랑주 승수의 의미

      • 라그랑주 승수법 $\rightarrow$ 등식 제한조건이 있는 최적화 문제를 푸는 방법
        • 등식 제한조건이란?
          • 현실의 최적화 문제는 보통 추가적인 제한조건이 포함됨
            • 제한조건
              • 등식 제한조건 : 입력변수의 값이 특정한 연립방정식을 만족
              • 부등식 제한조건 : 입력변수의 값이 특정한 연립부등식을 만족

    • 부등식 제한조건이 있는 최적화 문제의 KKT 조건

      • KKT 조건으로 부등식 제한조건을 품(?)

    • 사이파이(SciPy) 패키지를 사용한 제한조건이 있는 최적화 문제풀기

    • LP 문제와 QP 문제 : 최적화 문제 중 실용성이 높음

      • LP(linear programming) 문제 : 연립방정식의 해를 찾는 것
      • QP(quadratic programming) 문제 : LP를 약간의 수정해서 얻음

    확률분포

    • 사이파이를 이용한 확률분포 분석 방법
    • 이산확률분포 : 확률분포에는 범주값을 출력
      • 이산확률분포 정의 4가지
        1. 베르누이분포
        2. 이산분포
        3. 카테고리분포
        4. 다항분포
    • 연속확률분포 : 연속적인 값을 출력
      • 정규분포
        • 중심극한정리 : 실생활에서 정규분포가 많이 사용되는 이유를 알 수 있음
        • 분류문제로 활용을 보자 ! ex) 스팸 메일 필터링
        • 정규분포의 변형 3가지 : 데이터 간의 상관관계를 따지거나 회귀분석을 하기 위한 필수 - - 확률분포
          1. 스튜던트 t분포
          2. 카이제곱 분포
          3. F분포
        • 다변수 정규분포
    • 베이즈 추정의 개념

    • 엔트로피의 정의, 가변길이 인코딩과 엔트로피의 관계

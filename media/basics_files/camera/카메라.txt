
# 카메라 선택 방법과 기본기능

## 카메라 종류

### DSLR의 특징

- 광학식 뷰파인더
    - 장점 : 사실적인 사물 관찰
    - 단점 : 노출에 대한 이해 필요

- 지속적인 AF점검 필요 (초점이 안맞을 수 있음)
- 미러쇼크

### 미러리스의 특징

- 셔터가 항상 열려있음 (동영상 처럼)

- 전자식 뷰파인더
    - 장점 : 눈으로 보는 장면 그대로 촬영
    - 단점 : 세밀하게 피사체를 관찰하기 힘듬

- AF가 정확함
- 미러쇼크가 없음

## 사진 결정 3요소
사진의 밝기와 표현을 결정하는 3요소

조리개값,  셔터스피드값,  ISO감도는 상호유기적으로 영향을 미침

순서 예시 : ISO 감도 100dm으로 셋팅 - 조리개값 설정 - 셔터스피드값으로 조절하여 밝기를 맞춤


### 조리개값

빛의 양을 조절하는 장치

조리개 값이 작을수록 밝게나옴

#### 심도



### 셔터스피드값

1/250s 젤 빠르게 닫히는 속도 (한계)

빨리 닫힐수록 빛이 적게 들어옴

(셔터스프드가 느려지면 블러(흔들림) 발생)


### ISO감도

빛의 민감도 (숫자가 올라가면 밝게 찍히는 대신 노이즈가 발생)


## 측광

카메라가 빛을 측정하는 것

반자동모드에서 직접관여, 수동모드에서는 간접관여

"명부와 암부의 노출차를 줄이기 위한 기능이 아님"

종류 : 평가측광, 부분측광, 스팟측광

- 평가측광 : 눈으로 봤을 때, 고르게 보이는 상황에 적합
- 부분측광 : 인물사진에 많이 활용 (명부와 암부의 밝기 차이가 클 때 적합한 모드)
- 스팟측광 : 인물사진 및 세밀한 촬영에 많이 활용 (명부와 암부의 밝기 차이가 클 때 적합한 모드, 결혼식 부케 던질 때 등)


## AF (자동초점)

종류 (3가지)
- 위상차 방식 (DSLR) - 빠름 (부정확할 수 있음)
- 컨트라스트 방식 (구형 DSLR, 구형 미러리스 카메라) - 느리지만 정확함
- 하이브리드 방식 (위상차방식 + 컨트라스트방식) (최신 DSLR의 라이브뷰 촬영, 대부분의 미러리스 카메라) - 정확성과 속도 모두 챙김
